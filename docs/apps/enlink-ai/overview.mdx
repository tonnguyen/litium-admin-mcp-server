# EnlinkAI

*   Litium 8

The EnlinkAI app is used to integrate Litium with EnlinkAI's content creation platform, the AI-powered engine that simplifies, personalizes, and scales your content creation.

## Installation

Litium serverless cloud - Use the [Cloud CLI](https://docs.litium.com/cloud/serverless/get-started/install-payment-and-delivery-apps) to install the app.

Litium legacy cloud - Contact [support](mailto:support@litium.com) to install the app in your test and/or production environment.

Please note that you will need an API key from EnlinkAI to be able to use the app. Please reach out to EnlinkAI for more information.
