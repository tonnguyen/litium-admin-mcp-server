## Overview

Retrieve detailed information about a specific media file using its system ID.

<Info>
  This is a **read-only** tool that does not modify any data.
</Info>

## Parameters

<ParamField path="systemId" type="string" required>
  The unique system ID of the media file to retrieve
</ParamField>

## Response

Returns complete media file details including URL, file type, size, and metadata.

## Examples

<CodeGroup>
```bash cURL
curl -X POST https://your-server.vercel.app/api/mcp \
  -H "Content-Type: application/json" \
  -H "X-Litium-Base-Url: https://demoadmin.litium.com" \
  -H "X-Litium-Client-Id: your-client-id" \
  -H "X-Litium-Client-Secret: your-client-secret" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "get_media",
      "arguments": {"systemId": "media-123-456"}
    },
    "id": 1
  }'
```

```text Natural Language
"Show me the details of media file media-123-456"
```
</CodeGroup>

## Use Cases

- **View File Details**: Get file URL, size, and type
- **Verify Media**: Check if a media file exists and is accessible
- **Media Management**: Review file metadata before using in content
