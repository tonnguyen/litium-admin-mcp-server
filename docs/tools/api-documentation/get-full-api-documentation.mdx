## Overview

Get comprehensive API documentation including all endpoints, authentication details, examples, and usage patterns for the Litium REST API.

<Info>
  This is a **read-only** tool that provides complete API reference.
</Info>

## Parameters

No parameters required.

## Response

Returns complete documentation including:
- All API endpoints with details
- Authentication requirements
- Request/response examples
- Best practices
- Error handling
- Usage patterns

## Examples

<CodeGroup>
```bash cURL
curl -X POST https://your-server.vercel.app/api/mcp \
  -H "Content-Type: application/json" \
  -H "X-Litium-Base-Url: https://demoadmin.litium.com" \
  -H "X-Litium-Client-Id: your-client-id" \
  -H "X-Litium-Client-Secret: your-client-secret" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "get_full_api_documentation",
      "arguments": {}
    },
    "id": 1
  }'
```

```text Natural Language
"Show me the complete Litium API documentation"
```
</CodeGroup>
