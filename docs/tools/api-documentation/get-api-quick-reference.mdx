## Overview

Get a quick reference guide with common endpoints, authentication info, and base URL for the Litium API. Perfect for quick lookups and common operations.

<Info>
  This is a **read-only** tool that provides a condensed API reference.
</Info>

## Parameters

No parameters required.

## Response

Returns quick reference including:
- Common endpoints
- Authentication summary
- Base URL
- Most-used operations
- Quick examples

## Examples

<CodeGroup>
```bash cURL
curl -X POST https://your-server.vercel.app/api/mcp \
  -H "Content-Type: application/json" \
  -H "X-Litium-Base-Url: https://demoadmin.litium.com" \
  -H "X-Litium-Client-Id: your-client-id" \
  -H "X-Litium-Client-Secret: your-client-secret" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "get_api_quick_reference",
      "arguments": {}
    },
    "id": 1
  }'
```

```text Natural Language
"Show me a quick reference for the Litium API"
```
</CodeGroup>
