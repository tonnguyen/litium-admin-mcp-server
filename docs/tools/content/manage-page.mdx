## Overview

Manage pages. Operations: search, get, create, update, delete, unpublish, search_drafts, get_draft, create_draft, update_draft, delete_draft, publish_draft.

## Example

```json
{
  "name": "manage_page",
  "arguments": {
    "operation": "search",
    "params": {"search": "home"}
  }
}
```
