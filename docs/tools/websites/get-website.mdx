## Overview

Retrieve detailed configuration for a specific website using its system ID.

<Info>
  This is a **read-only** tool that does not modify any data.
</Info>

## Parameters

<ParamField path="systemId" type="string" required>
  The unique system ID of the website to retrieve
</ParamField>

## Response

Returns complete website configuration including domains, cultures, and settings.

## Examples

<CodeGroup>
```bash cURL
curl -X POST https://your-server.vercel.app/api/mcp \
  -H "Content-Type: application/json" \
  -H "X-Litium-Base-Url: https://demoadmin.litium.com" \
  -H "X-Litium-Client-Id: your-client-id" \
  -H "X-Litium-Client-Secret: your-client-secret" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "get_website",
      "arguments": {"systemId": "website-123"}
    },
    "id": 1
  }'
```

```text Natural Language
"Show me the configuration for website website-123"
```
</CodeGroup>

## Use Cases

- **Site Configuration**: View detailed website settings
- **Domain Verification**: Check configured domains and URLs
- **Culture Settings**: Review language and localization settings
