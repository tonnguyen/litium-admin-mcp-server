## Overview

Search for customers in your Litium instance using flexible search criteria. Perfect for customer support, order management, and account lookups.

<Info>
  This is a **read-only** tool that does not modify any data.
</Info>

## Parameters

<ParamField path="search" type="string" optional>
  Search term for filtering customers by name, email, or customer ID
</ParamField>

<ParamField path="skip" type="number" optional default="0">
  Number of items to skip for pagination
</ParamField>

<ParamField path="take" type="number" optional default="20">
  Number of items to return per page
</ParamField>

<ParamField path="sort" type="string" optional>
  Field to sort results by
</ParamField>

## Examples

<CodeGroup>
```bash cURL
curl -X POST https://your-server.vercel.app/api/mcp \
  -H "Content-Type: application/json" \
  -H "X-Litium-Base-Url: https://demoadmin.litium.com" \
  -H "X-Litium-Client-Id: your-client-id" \
  -H "X-Litium-Client-Secret: your-client-secret" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "search_customers",
      "arguments": {
        "search": "john@example.com"
      }
    },
    "id": 1
  }'
```

```text Natural Language (AI)
"Find the customer with email john@example.com"
```
</CodeGroup>

## Use Cases

- **Customer Support**: Find customer accounts quickly by email or name
- **Order Lookup**: Search for customers to find their orders
- **Account Management**: Locate customer profiles for updates
