## Overview

Retrieve detailed information about a specific customer using their system ID.

<Info>
  This is a **read-only** tool that does not modify any data.
</Info>

## Parameters

<ParamField path="systemId" type="string" required>
  The unique system ID of the customer to retrieve
</ParamField>

## Response

Returns complete customer details including contact information, addresses, and account status.

curl -X POST https://your-server.vercel.app/api/mcp \

  -H "X-Litium-Client-Id: your-client-id" \

"Show me the details for customer customer-123-456"

- **Customer Support**: View complete customer profile during support calls
- **Order Processing**: Get customer details when processing orders
- **Account Verification**: Confirm customer information
