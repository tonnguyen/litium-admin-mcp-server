## Overview

Manage customer accounts and profiles. Operations: search, get, create, update, delete.

## Operations

- `search` - Search for customers
- `get` - Get customer by ID
- `create` - Create new customer
- `update` - Update existing customer
- `delete` - Delete a customer

## Example

```json
{
  "name": "manage_customer",
  "arguments": {
    "operation": "search",
    "params": {
      "search": "john@example.com"
    }
  }
}
```

## Natural Language

"Find the customer with email john@example.com"
