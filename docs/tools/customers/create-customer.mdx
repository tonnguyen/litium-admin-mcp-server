## Overview

Create a new customer account in your Litium instance.

<Warning>
  This tool **modifies data** in Litium. Verify customer data before creating.
</Warning>

## Parameters

<ParamField path="customer" type="object" required>
  Complete customer data object
  
  <Expandable title="Customer Properties">
    <ParamField path="email" type="string" required>
      Customer email address (must be unique)
    </ParamField>
    
    <ParamField path="firstName" type="string" required>
      Customer first name
    </ParamField>
    
    <ParamField path="lastName" type="string" required>
      Customer last name
    </ParamField>
    
    <ParamField path="phone" type="string" optional>
      Phone number
    </ParamField>
  </Expandable>
</ParamField>

## Examples

<CodeGroup>
```bash cURL
curl -X POST https://your-server.vercel.app/api/mcp \
  -H "Content-Type: application/json" \
  -H "X-Litium-Base-Url: https://demoadmin.litium.com" \
  -H "X-Litium-Client-Id: your-client-id" \
  -H "X-Litium-Client-Secret: your-client-secret" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "create_customer",
      "arguments": {
        "customer": {
          "email": "newcustomer@example.com",
          "firstName": "John",
          "lastName": "Doe",
          "phone": "+1-555-0123"
        }
      }
    },
    "id": 1
  }'
```

```text Natural Language (AI)
"Create a new customer account for John Doe with email newcustomer@example.com"
```
</CodeGroup>

## Use Cases

- **New Account Creation**: Register new customers
- **Bulk Import**: Create customer accounts from external systems
- **Manual Registration**: Add customers who registered offline
