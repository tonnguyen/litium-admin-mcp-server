## Overview

The `manage_product` tool provides comprehensive management of base products in your Litium catalog. Use this tool to search, retrieve, create, update, and delete products, as well as get product variants.

## Operations

<ParamField path="operation" type="string" required>
  The operation to perform. Must be one of:
  - `search` - Search for products
  - `get` - Get a specific product by ID
  - `create` - Create a new product
  - `update` - Update an existing product
  - `delete` - Delete a product
  - `get_variants` - Get variants for a product
</ParamField>

## Parameters

### For `search` operation

<ParamField path="params" type="object">
  Search parameters:
  
  <Expandable title="properties">
    <ParamField path="search" type="string">
      Search term for filtering products
    </ParamField>
    <ParamField path="skip" type="number" default="0">
      Number of items to skip (pagination offset)
    </ParamField>
    <ParamField path="take" type="number" default="20">
      Number of items to return per page
    </ParamField>
    <ParamField path="sort" type="string">
      Sort field (optional)
    </ParamField>
  </Expandable>
</ParamField>

### For `get`, `update`, `delete`, `get_variants` operations

<ParamField path="systemId" type="string" required>
  The unique system ID of the product
</ParamField>

### For `create` and `update` operations

<ParamField path="data" type="object" required>
  Product data object with product fields
</ParamField>

## Examples

### Search Products

<CodeGroup>
```json JSON-RPC
{
  "jsonrpc": "2.0",
  "method": "tools/call",
  "params": {
    "name": "manage_product",
    "arguments": {
      "operation": "search",
      "params": {
        "search": "laptop",
        "take": 10
      }
    }
  },
  "id": 1
}
```

```text Natural Language
"Search for laptops in the product catalog and show the top 10 results"
```
</CodeGroup>

## Common Use Cases

<AccordionGroup>
  <Accordion title="Product Catalog Management">
    Use `search` to find products, `get` to view details, and `update` to modify pricing or descriptions.
  </Accordion>
  
  <Accordion title="Inventory Updates">
    Search for products by SKU, then update stock levels or pricing information.
  </Accordion>
  
  <Accordion title="Product Variants">
    Get a product and use `get_variants` to see all available sizes, colors, or other variations.
  </Accordion>
</AccordionGroup>
