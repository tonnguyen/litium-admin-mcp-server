## Overview

Update an existing product in your Litium e-commerce catalog. This tool allows you to modify product details, pricing, descriptions, and custom fields.

<Warning>
  This tool **modifies data** in Litium. Always verify the system ID and changes before updating.
</Warning>

## Parameters

<ParamField path="systemId" type="string" required>
  The unique system ID of the product to update
  
  **Example**: `"abc-123-def-456"`
</ParamField>

<ParamField path="product" type="object" required>
  Updated product data (only include fields you want to change)
  
  <Expandable title="Product Object Properties">
    <ParamField path="name" type="string" optional>
      Updated product name
    </ParamField>
    
    <ParamField path="description" type="string" optional>
      Updated description
    </ParamField>
    
    <ParamField path="price" type="number" optional>
      Updated price
    </ParamField>
    
    <ParamField path="fields" type="object" optional>
      Updated custom field values
    </ParamField>
  </Expandable>
</ParamField>

## Response

Returns the updated product with modifications applied.

<ResponseField name="systemId" type="string">
  System ID of the updated product
</ResponseField>

<ResponseField name="name" type="string">
  Updated product name
</ResponseField>

<ResponseField name="modified" type="string">
  Modification timestamp
</ResponseField>

## Examples

<CodeGroup>
```bash cURL
curl -X POST https://your-server.vercel.app/api/mcp \
  -H "Content-Type: application/json" \
  -H "X-Litium-Base-Url: https://demoadmin.litium.com" \
  -H "X-Litium-Client-Id: your-client-id" \
  -H "X-Litium-Client-Secret: your-client-secret" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "update_product",
      "arguments": {
        "systemId": "abc-123-def-456",
        "product": {
          "price": 999.99,
          "fields": {
            "onSale": true,
            "saleEndDate": "2024-12-31"
          }
        }
      }
    },
    "id": 1
  }'
```

```text Natural Language (AI)
"Update product abc-123-def-456 to sale price $999.99"
```
</CodeGroup>
