## Overview

Retrieve detailed information about a specific product using its system ID. This tool uses the Litium Admin API endpoint `GET /Litium/api/admin/products/baseProducts/{systemId}`.

<Info>
  This is a **read-only** tool that does not modify any data.
</Info>

## Parameters

<ParamField path="systemId" type="string" required>
  The unique system ID of the product to retrieve
  
  **Example**: `"abc-123-def-456"`, `"product-uuid-12345"`
</ParamField>

## Response

Returns complete details of the requested product.

<ResponseField name="systemId" type="string">
  Unique identifier for the product
</ResponseField>

<ResponseField name="id" type="string">
  Product ID/SKU
</ResponseField>

<ResponseField name="name" type="string">
  Product name
</ResponseField>

<ResponseField name="description" type="string">
  Full product description
</ResponseField>

<ResponseField name="price" type="object">
  Pricing information including base price, sale price, currency
</ResponseField>

<ResponseField name="stock" type="object">
  Inventory levels and availability
</ResponseField>

<ResponseField name="images" type="array">
  Product images and media
</ResponseField>

<ResponseField name="categories" type="array">
  Category assignments
</ResponseField>

<ResponseField name="variants" type="array">
  Product variants (if applicable)
</ResponseField>

## Examples

<CodeGroup>
```bash cURL
curl -X POST https://your-server.vercel.app/api/mcp \
  -H "Content-Type: application/json" \
  -H "X-Litium-Base-Url: https://demoadmin.litium.com" \
  -H "X-Litium-Client-Id: your-client-id" \
  -H "X-Litium-Client-Secret: your-client-secret" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "get_product",
      "arguments": {
        "systemId": "abc-123-def-456"
      }
    },
    "id": 1
  }'
```

```text Natural Language (AI)
"Show me the details of product abc-123-def-456"
```
</CodeGroup>
