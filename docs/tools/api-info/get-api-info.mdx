## Overview

The `get_api_info` tool provides access to Litium API documentation and endpoint discovery. Use this tool to explore available API endpoints, understand OAuth configuration, and get quick API references.

## Operations (Types)

<ParamField path="type" type="string" required>
  The type of API information to retrieve:
  - `full` - Complete API documentation with all endpoints and schemas
  - `quick` - Quick reference guide with endpoint summaries
  - `oauth` - OAuth2 configuration and authentication information
  - `endpoints` - List endpoints filtered by category
</ParamField>

## Parameters

### For `endpoints` type

<ParamField path="category" type="string">
  Filter endpoints by category (e.g., "Products", "Content", "Sales", "Media")
</ParamField>

<ParamField path="search" type="string">
  Search endpoints by keyword or path
</ParamField>

## API Categories

The Litium Admin API is organized into these main categories:

- **Products** - Base products, variants, categories, assortments, price lists
- **Content** - Blocks, pages, websites
- **Media** - Files, folders, media assets
- **Sales** - Orders, order history
- **Customers** - Customer accounts and profiles
- **Globalization** - Channels, markets, currencies, languages, countries
