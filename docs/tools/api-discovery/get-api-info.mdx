## Overview

Get general information about the Litium Admin API including base URL, authentication method, content type, and common headers.

<Info>
  This is a **read-only** discovery tool.
</Info>

## Parameters

No parameters required.

## Response

Returns general API configuration including:
- Base URL
- Authentication method (OAuth2)
- Content-Type requirements
- Common headers
- API version information

## Examples

<CodeGroup>
```bash cURL
curl -X POST https://your-server.vercel.app/api/mcp \
  -H "Content-Type: application/json" \
  -H "X-Litium-Base-Url: https://demoadmin.litium.com" \
  -H "X-Litium-Client-Id: your-client-id" \
  -H "X-Litium-Client-Secret: your-client-secret" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "get_api_info",
      "arguments": {}
    },
    "id": 1
  }'
```

```text Natural Language
"Show me general Litium API information"
```
</CodeGroup>
