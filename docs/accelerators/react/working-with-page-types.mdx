# Working with page types

The page types in the React Accelerator is defined by the associated field templates. You can find the field templates in the Settings area in Backoffice, in the Website section.

The following page types is included in the Litium React Accelerator:

- [Article page](/accelerators/react/working-with-page-types/article-page)
- Category page
- Product list page
- Product page
- Search result page
