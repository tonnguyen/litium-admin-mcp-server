# Article page

The article page type is suited for the editorial content of your storefront, eg product guides, contact information and blog posts.

The article page has the following fields:

| Field name | Field type   | Description |
| --- | --- | --- |
| Name | Text | The name of the page, used in the left menu |
| URL | Text | The URL of the page |
| Show navigation for child pages | Boolean | if set to true, the left menu will display a tree structure for any child pages of the current page. |
| Title | Text | The title of the page, used as the header when the page is displayed |
| Introduction | MultirowText | Short introduction to the contents of the page |
| Text | Editor | The content of the page |

The article page has one block container.
