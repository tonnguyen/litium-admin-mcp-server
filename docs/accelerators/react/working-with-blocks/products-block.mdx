# Products block

The products block is suited for displaying a selection of products on pages of your website. Products to show can be selected from a category or a product list.

The product block has the following fields:

| Field name | Field type | Description |
| --- | --- | --- |
| Name | Text | The name of the block |
| Title | Text | The title of the block |
| Category | Pointer | Category to display products from |
| Product list | Pointer | Product list to display products from. Will only be used if no category is selected. |
| Number of products | Integer | Number of products to display. If no value is entered, a default number of 15 products will be displayed. A maximum number of 100 products will be displayed. |
| Sorting | Text option | Possible values: Recommended or Popular. Recommended will display only products from the selected category, not sub-categories. Popular will display products from the selected category including subcategories. |
