# Settings

In the website fields template, there are pointer fields designated to set up the URL for the pages, which are then utilized to retrieve pages in the React accelerator.

![Website settings](/images/accelerators/react/website-settings.png)

- Field group **Search**:  
  Search result page: The page where the user ends up after performing a search
- Field group **Order confirmation page**:  
  Order confirmation page: The page that will be displayed as order confirmation page
- Field group **Errors**:  
  Page not found (404): The pointed out page will be shown to users when a page is not found.  
  General error (500): The pointed out page will be shown to users when the system returns a general 500-error

**Notes:** In a 500 error page, only the title and information about the error are displayed, no blocks are displayed.
