# What's new in 8.1

Litium 8.1 is a minor version of the Litium commerce platform. It contains minor features and bug fixes.

##

## Highlights

- New discount type: Buy x products, get discount on cheapest
- New discount type: Mix and match
- New discount type: Free gift
- Additional info on orders and order rows
- Allow custom authentication (like 2-factor authentication)

## New discount type: Buy x products, get discount on cheapest

The ability to use discounts where buyers get to buy x number of products and get a discount on the cheapest product.

## New discount type: Mix and match

The ability to use discounts where buyers will get one special price when buying x number of products from a given set of products.

## New discount type: Free gift

The ability to use discounts where buyers will a get free gift.

## Additional info on orders and order rows

Enable for projects to store additional info on orders and order rows to be able to customize orders in a flexible way. Please read how to work with AdditionalInfo entity.

## Allow custom authentication (like 2 factor authentication)

The ability to authenticate to the Litium administration interface with existing authentication that is in use in an organization, for example, Active Directory or Azure AD. The custom external authentication that is set up in the project can be configured to ensure multi-factor authentication. Please see a sample on Github.
