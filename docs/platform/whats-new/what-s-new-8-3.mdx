# What's new in 8.3

Litium 8.3 is a minor version of the Litium commerce platform. It contains minor features and bug fixes.

## Highlights

- Ability to sort products on most sold
- Ability to manage multiple generated discount codes
- Improvements in Connect Payments API
- Ability for partner to setup a test project
- Demo site improvements
- Accelerator mega menu redesigned

## Ability to sort products on most sold

The ability to sort products according to how much sales they have generated.

## Ability to manage multiple generated discount codes

With this feature it is possible to have the system generate random codes that can be used to grant a discount.

## Improvements in Connect Payments API

More robust API to handle asynchronous communication and ability for the paymnet apps to communicate supported operations for each payment.

## Ability for partner to setup test project

A test framework that makes it easier for partners to write unit tests for the Litium Accelerator.

## Demo site improvements

The demo site is updated with more content including preconfigured discounts, campaigns, blocks of all block types, more pages showing campaigns, brands and corporate content, as well as workflows and tasks in PIM.

## Accelerator mega menu redesigned

The mega menu of the Litium Accelerator has been given a light face lift.
