### API changes from version to version of Litium

- Litium 7 to 8: The Sales API is re-written.
- Litium 6 to 7: The Websites API is re-written.
- Litium 5 to 6: The Customers and Media API are re-written.
- Litium 4 to 5: The Products API is re-written.

For more information [information about changes in the domain model between Litium 5 and Litium 7, please see this table](/platform/previous-versions/litium-7/upgrading-to-litium-7/api-changes) in the Litium 7 upgrade documentation. Litium 8 comes with a new order management API, new campaign management API, new Litium Connect Payments API, and new Litium Connect Shipment API.
