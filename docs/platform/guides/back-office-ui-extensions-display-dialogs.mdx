This section shows how to use dialogs in the Litium backoffice.

In Litium 8 all dialogs are built with Angular.

- [Dialogs using Angular](#dialogs-using-angular)
- [Universal dialogs](#universal-dialogs)

## Dialogs using Angular

To open a dialog in the backoffice and select a value, add the following to your component template:

```html
<a class="base__fake__link" (click)="showDialog = true">
  {{ 'general.action.add' | translate }}
</a>
<entity-selector-dialog
  *ngIf="showDialog"
  [entityType]="'group'"
  [entityMultiSelect]="false"
  (onSubmit)="submit($event)"
  (onClose)="showDialog = false">
</entity-selector-dialog>
```

Add the corresponding code to the component:

```ts
showDialog = false;

submit(ids: string[]) {
  // Process the selected IDs
}
```

The anchor renders an **Add** link that opens the entity selector dialog. The example lists groups and allows selecting a single group.

Options:

- `entityType`: the entity to select (see the list below).
- `entityMultiSelect`: whether a user can pick multiple items (default `true`).
- `onSubmit`: callback for the OK button. Receives the selected IDs.
- `onClose`: callback for the Cancel button. Set `showDialog` to `false` to close the dialog.

## Universal dialogs

The Angular approach above runs within the Angular runtime. When that is not possible you can open dialogs through an iframe. Universal dialogs support all entity types listed above plus media files and pages.

The idea is to render an iframe and listen for messages when a user selects an item. Use one of the following URLs as the iframe `src`:

- **Media file**  
  `https://localhost/Litium/UI/frame/FileSelectorWrapper?multiSelect=true&type=4`

  - Inputs:
    - `multiSelect`: `true` enables multi-select.
    - `type`: `1` (all files, default), `2` (images), or `4` (video files).
  - Output:
    - `source`: `fileSelector`
    - `value`: `{ action, items }` where `action` is `closeMediaPopup` (OK/Cancel) or `doubleClick`, and `items` contains the selected files.

- **Page**  
  `https://localhost/Litium/UI/frame/EntityTreeSelectorWrapper?entityMultiSelect=1&entityType=page`

  - Input:
    - `entityMultiSelect`: `0` or `1`
  - Output:
    - `source`: `entitytreeselector`
    - `value`: `{ items }` (array if multi-select, otherwise an object with `systemId` and `channelSystemId`)

- **Category**  
  `https://localhost/Litium/UI/frame/EntityTreeSelectorWrapper?entityMultiSelect=1&entityType=category`

  - Output:
    - `value.items` contains the selected categories (`systemId` only when single select)

- **Other entity types**  
  `https://localhost/Litium/UI/frame/EntitySelectorWrapper?entityMultiSelect=1&entityType=countries`

  - Inputs:
    - `entityMultiSelect`: `0` or `1`
    - `entityType`: one of the names from the table above
  - Output:
    - `source`: `entityselector`
    - `value.items`: array of `{ name, systemId }`

Example snippet:

```html
<html>
  <body>
    <script type="text/javascript">
      window.addEventListener('message', receiveMessage, false);

      function receiveMessage(event) {
        // Handle the response from the iframe
        // event.data is the output object described above
      }
    </script>

    <iframe src="https://localhost/Litium/UI/frame/EntitySelectorWrapper?entityMultiSelect=1&entityType=countries"></iframe>
  </body>
</html>
```
