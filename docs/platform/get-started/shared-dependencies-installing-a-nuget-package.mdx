Methods to install NuGet packages.

### Install the package from NuGet Package Manager

1.  Open NuGet Package Manager
2.  Find  your package
3.  Select the project to install into and select **Install**

**​**

Please see [additional ways to install a NuGet package here](https://docs.microsoft.com/en-us/nuget/consume-packages/overview-and-workflow#ways-to-install-a-nuget-package) and [_PackageReference_ in project files here](https://docs.microsoft.com/en-us/nuget/consume-packages/package-references-in-project-files).
