description: "To generate Excel texts file from .yaml files, execute the following command in Powershell command prompt:"
---
To generate Excel texts file from .yaml files, execute the following command in Powershell command prompt:

litium-storefront text convert-to-excel -i '../Litium.Storefront/litium-definitions/texts/\*.yaml' -f '../Litium.Storefront/texts.xlsx'

The command will create an excel file named texts.xlsx under Litium.Storefront folder.
