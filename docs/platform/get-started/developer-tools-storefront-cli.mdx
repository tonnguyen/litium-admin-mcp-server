This page is an introduction to the Storefront CLI.

## Install Storefront CLI

**Prerequirements**

- Docs account, created via [Docs](https://docs.litium.com/system_pages/login_ls)
- [NET Core 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
- [Litium nuget feed](/platform/get-started/litium-packages)

To install or update the tool, execute the following command in Powershell command prompt:

```powershell
dotnet tool update -g Litium.Storefront.Cli
```
