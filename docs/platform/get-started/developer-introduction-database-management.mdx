---
title: "Database management"
description: "Litium requires a Microsoft SQL Server 2019 database."
---
Litium [requires a Microsoft SQL Server 2019 database](/platform/guides/upgrade-to-litium-8/upgrade-the-platform-solution/dotnet-upgrade).

The command-line donet tool **litium-db** is used to manage the database, it can be used to:

- Setup new Litium application databases
- Upgrade a Litium application database to a specific version, either by connecting directly to the database or by generating scripts for later upgrade
- [Create new admin users](/platform/get-started/create-admin-user-and-log-in) in the database
