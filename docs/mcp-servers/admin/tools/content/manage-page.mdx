## Overview

The `manage_page` tool provides comprehensive management of website pages in Litium, including both published pages and draft versions. Use this tool for full page lifecycle management.

<Note>
  **Enhanced Search:** Page search now searches across multiple fields (ID, Title) for better discovery of pages.
</Note>

## Operations

<ParamField path="operation" type="string" required>
  The operation to perform:
  
  **Published Pages:**
  - `search` - Search for published pages
  - `get` - Get a published page by ID
  - `create` - Create and publish a new page
  - `update` - Update a published page
  - `delete` - Delete a published page
  - `unpublish` - Unpublish a page
  
  **Draft Management:**
  - `search_drafts` - Search for page drafts
  - `get_draft` - Get a draft by ID
  - `create_draft` - Create a new draft
  - `update_draft` - Update an existing draft
  - `delete_draft` - Delete a draft
  - `publish_draft` - Publish a draft to make it live
</ParamField>

## Parameters

### For `search` and `search_drafts` operations

<ParamField path="params" type="object">
  Search parameters (all optional):
  
  <Expandable title="properties">
    <ParamField path="search" type="string">
      Search term for filtering pages
    </ParamField>
    <ParamField path="skip" type="number" default="0">
      Pagination offset
    </ParamField>
    <ParamField path="take" type="number" default="20">
      Results per page
    </ParamField>
    <ParamField path="sort" type="string">
      Sort field
    </ParamField>
  </Expandable>
</ParamField>

### For `get`, `update`, `delete`, `unpublish` operations

<ParamField path="systemId" type="string" required>
  The unique system ID of the page
</ParamField>

### For `create`, `update`, `create_draft`, `update_draft` operations

<ParamField path="data" type="object" required>
  Page data object with page fields and content
</ParamField>

## Common Workflows

<AccordionGroup>
  <Accordion title="Draft-to-Publish Workflow">
    1. Create draft: `create_draft`
    2. Review draft: `get_draft`
    3. Update draft: `update_draft` (if needed)
    4. Publish: `publish_draft`
  </Accordion>
  
  <Accordion title="Content Update Workflow">
    1. Find page: `search`
    2. Get details: `get`
    3. Update content: `update`
    4. Verify: `get` again
  </Accordion>
  
  <Accordion title="Page Retirement">
    1. Find page: `search`
    2. Unpublish: `unpublish` (keeps page but removes from site)
    3. Or delete: `delete` (permanently removes)
  </Accordion>
</AccordionGroup>
