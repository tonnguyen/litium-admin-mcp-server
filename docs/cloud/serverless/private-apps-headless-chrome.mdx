## Installation

**Prerequisites**

Make sure the following are in place:

- [CLI installed](https://docs.litium.com/platform/get-started/developer-tools/litium-cloud-cli/getting-started-with-cloud-cli)
- [Access to the subscription where you want to create a new environment](https://docs.litium.com/platform/get-started/developer-tools/litium-cloud-cli/getting-started-with-cloud-cli)
- [Environment created](/cloud/serverless/create-environment-new)

### Step 1: Download the Manifest

Use the following command to download the Headless Chrome manifest:

```powershell
litium-cloud marketplace manifest --app headless-chrome -f enter-any-name.yaml
```

```powershell

  type: headless-chrome
  plan: [insert-plan-id]
  version: 1.0.0
```

## How to use

To connect to your Headless Chrome instance from another app (e.g., Litium platform or a .NET app), use the WebSocket **URL** exposed by the Headless Chrome app. 

**Note** - The exposed URL is internal and can only be accessed by other apps within **Serverless Cloud**. It is not accessible from outside the Serverless Cloud environment.

**Tip** - View properties that an app exposes  
litium-cloud marketplace show --app headless-chrome --version 1.0.0

### Example: Configure another app to use Chrome

To use the Chrome socket in another app, configure it to fetch the URL by setting up configurations:

```powershell
  configurations:
    - name: CHROME__SOCKET
      valueFrom:
        appRef:
          name: headless-chrome
          key: url
```

This will populate the `CHROME__SOCKET` environment variable with the WebSocket URL from the Headless Chrome app.

Add this configuration section to the YAML file of the app that needs to access Headless Chromeâ€”such as the Litium platform appâ€”then re-apply it using "apply" command.

### Example - generate PDF with PuppeteerSharp

Below is a sample C# method using [PuppeteerSharp](https://github.com/hardkoded/puppeteer-sharp) to generate a PDF:

```csharp
public async Task<byte[]> GeneratePdf()
{
    string htmlContent = @"<!DOCTYPE html>
                        <html lang=""en"">
                        <head>
                        </head>
                        <body>
                            <h1>Test PDF</h1>
                            <p>This is a test PDF document generated using PuppeteerSharp.</p>
                        </body>
                        </html>
                        ";


    var browserWSEndpoint = Environment.GetEnvironmentVariable("CHROME__SOCKET");

    var connectOptions = new ConnectOptions
    {
        BrowserWSEndpoint = browserWSEndpoint,
        DefaultViewport = null
    };

    var browser = await Puppeteer.ConnectAsync(connectOptions);

    using (var page = await browser.NewPageAsync())
    {
        // Set content of the page
        await page.SetContentAsync(htmlContent);

        // Generate PDF
        return await page.PdfDataAsync(new PdfOptions
        {
            Format = PaperFormat.A4,
            PrintBackground = true,
            MarginOptions = new MarginOptions
            {
                Top = "20mm",
                Bottom = "20mm",
                Left = "15mm",
                Right = "15mm"
            }
        });
    }
}
```
