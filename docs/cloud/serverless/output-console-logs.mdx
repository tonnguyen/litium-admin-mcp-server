For certain apps it's possible to get console output generated to logs.

**Prerequirements**

- An app that has the action "console-output" available - currently available for Litium platform, Litium Next.js, Next.js web, Nuxt.js web, Node.js web. .NET web

## Execute action

This action is especially useful when your app doesn't start for some reason, and logs isn't sent to Litium Insights because your app haven't started. In other "normal" cases this action shouldn't be needed because your logs are collected and displayed in Litium Insights.  

Example action command:

```powershell
litium-cloud app action --action console-output --app litium
```

_Note. above command assumes that you have an app with id 'litium' and that you have set subscription and enviornment to the context_

This action results in a job where the logs are collected. 

1.  Execute status show command to get id of related console-output job, litium-cloud status show --job bb5d847eaehidden
