Private apps makes it possible to manage other workloads that is related to the e-commerce ecosystem. For instance integration apps, private databases, additional storage and more.

Private apps offer a flexible solution tailored to your specific needs, acting as customizable slots in your environment for various workloads or data. Each app serves a unique purpose, such as storage, database management, or hosting .NET web containers. You can install multiple private apps, selecting an App plan for each that defines its resource allocation. Pricing for private apps is hourly and varies by the chosen App plan. Discover the diverse applications and their pricing in the sections below.

## [.NET web](/cloud/serverless/private-apps-dotnet-web)

Deploy custom .NET web artifacts, integrating services like file storage, Redis cache, and MS SQL databases. Ideal for periodic custom workloads or creating advanced web pages, such as customer review management systems. Select from various App plans to allocate resources appropriately, with costs based on the chosen plan.

## [File storage](https://docs.litium.com/platform/get-started/developer-tools/litium-cloud-cli/private-apps/file-storage)

Use this app for storing files, such as integration documents. It seamlessly integrates with apps like Litium and SFTP for syncing ERP data, with charges based on disk usage.

## [Headless Chrome](/cloud/serverless/private-apps-headless-chrome)

The Headless Chrome app gives you access to a Chrome browser for automation tasks such as generating PDFs from HTML content, capturing screenshots, running automated UI tests, and more. It exposes a WebSocket URL that can be consumed by other applicationsâ€”such as the Litium platform or a .NET web app.

## [MS SQL database](/cloud/serverless/private-apps-ms-sql)

Host your MS SQL databases, offering easy integration with the .NET Web app. Pricing is determined by disk usage.

## [Next.js web](/cloud/serverless/private-apps-nextjs-web)

Host your private Next.js web app if you for example want additional storefronts connected to your Litium app.

## [Node.js Web](/cloud/serverless/private-apps-nodejs-web)

In Litium Serverless Cloud, you can run Node.js applications without the need for a specific framework like Next.js or Nuxt.js. Simply use the common nodejs-web app, which supports any type of Node.js application. The app relies on the build and start commands defined in the package.json file to manage the application's lifecycle.

## [Nuxt.js web](/cloud/serverless/private-apps-nuxt-js-web)

Host your private Nuxt.js web app to host your Vue.js storefronts connected to your Litium app.

## [Redis Cache](/cloud/serverless/private-apps-redis-cache)

Enhance access speed with a private Redis Cache, connectable to .NET Web apps for efficient data retrieval. Costs are App plan-dependent.
